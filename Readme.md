# LangSwitcher

Простое приложение для Windows 10, которое переключает раскладку по отдельным Shift-клавишам:
- `Left Shift` -> English (`en`)
- `Right Shift` -> Russian (`ru`)

Текущая версия: `0.0.1`

## Зависимости

- Python 3.12+
- Библиотеки из `requirements-dev.txt`

Установка:

```bash
python -m venv .venv
./.venv/bin/pip install -r requirements-dev.txt
```

## Запуск

```bash
./.venv/bin/python main.py
```

Остановка приложения: нажмите `Esc` в активной сессии (по умолчанию listener ждет этот сигнал завершения).

Можно передать свой конфиг:

```bash
./.venv/bin/python main.py --config /path/to/config.yaml
```

## Формат конфига

По умолчанию используется `config/default_config.yaml`:

```yaml
log_level: INFO
hotkeys:
  left_shift: en
  right_shift: ru
```

`log_level` поддерживает значения: `TRACE`, `DEBUG`, `INFO`, `WARNING`, `ERROR`, `CRITICAL`.
Рекомендуемый стартовый уровень: `INFO`.

Допустимые ключи:
- `left_shift`
- `right_shift`

Допустимые раскладки:
- `en`
- `ru`

## Автотесты

```bash
./.venv/bin/pytest -q
```

## Логирование

- Логирование выполняется через `loguru` в терминал.
- На старте приложение пишет путь до конфига, примененный уровень логирования и активную карту hotkey.
- При `log_level: DEBUG` дополнительно выводятся события нажатий Shift и попытки/результаты переключения раскладки.

## Ограничения Windows 10

- Приложение ориентировано на Windows 10.
- Для глобального перехвата клавиш используются системные хуки: в некоторых окружениях могут потребоваться дополнительные права.
- Если другое ПО уже перехватывает Shift, поведение может зависеть от порядка обработки системных событий.
- Запуск из WSL (Linux Python) не поддерживается: в этом режиме недоступны нужные Windows API и глобальные хуки клавиатуры.
- Если вы работаете в WSL, запускайте приложение через интерпретатор Python в Windows (PowerShell/cmd), а не через `python` внутри WSL.

## Почему используются эти библиотеки

- `yamldataclassconfig`: удобная загрузка YAML в dataclass-модель, чтобы не писать лишний ручной парсинг.
- `keyboard`: популярная библиотека для глобального прослушивания клавиш в Windows.
- `pytest`: минимальный и стандартный инструмент для автотестов.
